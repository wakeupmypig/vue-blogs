# 从零基础到实战 全面解析uni-app (一)

# 1.什么是uni-app

uni-app全称是Union Application,可以通过编写一套代码布到iOS、Android、H5、以及各种小程序平台

uni-app特点:

  • 接近原生app体验，支持多平台
  • 开发效率高，开发成本低
  • 学习成本低,uni-app是基于vue框架的
  • 方便扩展，周边生态丰富

# 2.安装开发工具-HBuilderX

安装 HBuilderX，根据系统下载对应的安装包

下载后创建项目: 文件 -> 新建 -> 项目 -> uni-app -> 默认模板

到此项目就创建成功啦~~

# 3.运行项目

可以在浏览器中访问，也可以直接使用真机调试（需要将手机连接至电脑）

需要设备信任:设置 -> 通用 -> 设备管理 -> 信任

# 4.在小程序中运行

要使用微信小程序开发工具预览，需要先下载微信开发者工具,启动小程序安全设置中的端口。并将ide路径粘贴至运行设置中

这样就可以运行到小程序模拟器中啦~

# 5.在mac模拟器运行

使用mac系统，我们可以直接通过安装xcode来获取Simulor模拟器，运行代码点击运行选择

可以直接启动模拟器会自动运行代码~~~
