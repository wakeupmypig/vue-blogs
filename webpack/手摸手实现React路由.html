# 手摸手实现React路由

现在SPA应用已经非常的普及了,当然路由的实现我们也必须要掌握啦~

经常面试会遇到:你了解前端路由吗?

前端路由共有两种实现方式:

  • Hash实现前端路由
  • H5 HistoryApi实现前端路由

# 1.简介前端路由

# 1.1 hash

hash路由最明显的特征是url路径后面有 #,通过hash值的变化来控制显示到页面的内容,优点是兼容性好，缺点是不够美观,在开发环境下我们可能会选择hash路由

<a href="#/">首页</a>
<a href="#/about">关于页面</a>
<div id="content"></div>
<script>
  window.location.hash = window.location.hash.slice(1) || '/';
    content.innerHTML = window.location.hash; // 初始化默认路径
    window.addEventListener('hashchange',()=>{ // 监听hash值的变化
    content.innerHTML = window.location.hash;
  })
</script>

# 1.2 History API

history.pushState用于在浏览历史中添加历史记录,但是并不触发跳转,此方法接受三个参数，依次为：state,title,url

<a onclick="push('/')">首页</a>
<a onclick="push('/about')">关于页面</a>
<div id="content"></div>
<script>
function push(path){
  history.pushState(null,{},path);
  content.innerHTML = window.location.pathname;
}
content.innerHTML = window.location.pathname; // 初始化默认内容
window.addEventListener('popstate',function(){ // 监听浏览器前进后退的事件
  content.innerHTML = window.location.pathname;
})
</script>

# 2.2 HashRouter实现

HashRouter组件的主要功能就是：监听路由变化，更新路由信息的相关状态,将数据传递给后代组件

import React from 'react';
import Context from './context';
export default class HashRouter extends React.Component{
    state = {
        location:{
            pathname:window.location.hash.slice(1),
        }
    }
    componentDidMount(){
        // 页面加载如果没有hash 默认跳转到/
        window.location.hash = window.location.hash.slice(1) || '/';
        window.addEventListener('hashchange',()=>{
            this.setState({ // 当路由变化时 跳转到指定路径
                location:{
                    ...this.state.location,
                    pathname:window.location.hash.slice(1)
                }
            })
        });
    }
    render(){
        let props = { // 将props传递给所有后代组件
            location:this.state.location
        }
        return <Context.Provider value={props}>
            {this.props.children}
        </Context.Provider>
    }
}

# 2.3 Route组件实现

import React from 'react';
import context from './context';
export default class Route extends React.Component{
    static contextType = context;
    render(){
        // 获取当前请求的路径
        let pathname = this.context.location.pathname
        // 获取当前配置的路径和组件
        let {path,component:Component}  = this.props;
        // 路径相等则渲染对应的组件
        if(pathname === path){
            return <Component/>
        }
        return null;
    }
}
